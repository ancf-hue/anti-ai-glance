Анти ИИ Взгляд – помощь слепым и слабовидящим без интернета

Краткое описание:

Веб-приложение, которое превращает обычный смартфон в тактильно-голосовой детектор препятствий, ям и неровностей. Работает полностью офлайн, без интернета, без нейросетей, на любом современном телефоне с камерой.

Принцип работы:

Анализирует нижнюю треть кадра (зона перед ногами).

Делит на сетку 4×4 (16 ячеек).

Сравнивает текущую яркость с эталоном.

Фильтр тряски: если изменилось ≥10 ячеек — игнорируем (это шаг или поворот телефона).

Локальное изменение 1–9 ячеек, подтверждённое в 2+ кадрах → голос «Опасность» + вибрация.

Дополнительно акселерометр ловит резкий провал («Провал!»).

Как использовать:

Скачайте index.html или откройте [ссылка на GitHub Pages].

Нажмите «Запустить помощника», разрешите доступ к камере.

Направьте телефон чуть вниз, чтобы видеть дорогу на 1–2 метра.

Идите медленно. Система будет молчать на ровном пути.

Поднесите руку к камере → услышите «Опасность».

Требования:

Android 8+ / Chrome (минимальные требования: 1 ГГц, 1 ГБ ОЗУ).

Камера с фокусом от 30 см.

Как помочь проекту:

Тестировать на разных телефонах и в разных условиях (снег, дождь, грунтовка).

Улучшать алгоритм – портировать на нативное приложение (Kotlin/Swift), добавить опциональный TensorFlow Lite.

Сделать голосовое управление (офлайн-команды через Vosk).

Перевести на другие языки.

Добавить поддержку жестов (двойной тап – сброс эталона, долгий тап – вспышка).

Лицензия: MIT (открытая, можно использовать в любых проектах).
